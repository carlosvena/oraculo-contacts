# Flujo de revisión

La aplicación guía cinco etapas: importar, analizar, revisar, previsualizar y exportar. Cada propuesta
muestra antes/después, evidencia, confianza, riesgo, regla e impacto. Aprobar, rechazar o posponer crea
un nuevo `ChangeSet`; el contacto importado permanece intacto.

La aprobación masiva acepta únicamente propuestas de bajo riesgo y excluye campos protegidos. Las
decisiones pueden deshacerse, rehacerse o restaurarse durante la sesión.
